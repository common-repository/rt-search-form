== Description ==

This plugin add search form in post/page using shortcode [rtsearchform].

### Features
* Use [rtsearchform] sortcode to call search form inside post.

== Installation ==

= Install =

1. Upload `rt-search-form` folder to the `/wp-content/plugins/` directory
2. In your WordPress administration, go to the Plugins page
3. Activate the Royal Search Form.
